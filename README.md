# 🐍 Python + Cursor: Desenvolvimento Inteligente com IA

Repositório oficial do curso **Python + Cursor: Smarter Development with AI** da Santander Open Academy.

Aprenda a programar **mais rápido e de forma mais inteligente** usando o Cursor, o editor de código impulsionado por inteligência artificial.

## ✨ Sobre o curso

Este curso gratuito ensina como aproveitar ao máximo o **Cursor** (um IDE baseado em IA) para desenvolver aplicações em Python de forma muito mais produtiva.

Você vai aprender desde a instalação até fluxos avançados, incluindo:
- Interface e recursos inteligentes do Cursor
- Desenvolvimento Python com sugestões de IA em tempo real
- Refatoração automática e depuração inteligente
- Integração com Git
- Uso de Docker para implantação
- Frameworks como Flask e Django
- Um projeto prático ao final

**Duração:** 8 horas (no seu ritmo)  
**Nível:** Intermediário (recomendado ter conhecimentos básicos de Python)  
**Formato:** 100% online e gratuito  
**Certificado:** Disponível ao concluir

## 🎯 O que você vai aprender

- Instalar e configurar o Cursor
- Dominar prompts e comandos de IA para codar mais rápido
- Criar projetos completos com ajuda da IA
- Boas práticas de desenvolvimento moderno com IA
- Deploy e integração de ferramentas
